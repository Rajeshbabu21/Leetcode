Input: arr[] = [1, 2, 3, 5]
Output: 4
Explanation: Here the size of the array is 4, so the range will be [1, 5]. The missing number between 1 to 5 is 4

class Solution {
    public int Missingnumber(int[] nums) {

        👉 The array is missing one number
        👉 So actual full range has one extra element
        int n = nums.length + 1;

        int sum = 0;
        for(i=1;i<=n;i++)
        {
            sum+=i;
        }
        int arrsum = 0;
        for(int i:nums)
        {
            arrsum+=i;
        }

        return sum - arrsum;
        
    }
}
